SAE 4.01 - Développement d'une API


Pour ce projet que j'ai réalisé avec Victor Dhorme. Nous avons dû créer une API pour une pizzeria fictive.

Visuel d'un résultat d'une requête


Description

Nous avons eu comme objectif de créer une API qui permet de voir, créer, modifier des données qui serait utile pour une pizzeria.

Les données étant stockées dans une base de données, l'enjeu était de faire la passerelle entre la requête et la base de données.

Nous avons utilisé le langage Java (Servlet) pour le développement de l'API.

Nous avons dû également mettre un système d'authentification pour effectuer certaines requêtes comme la modification, création, suppression des données

Date

Projet réalisé en février 2024 (2 ème année de B.U.T. Informatique)

Compétences

  • Travail en équipe
  • Création d'une base SQL
  • Interprétation des requêtes reçues
  • Respecter un protocole
