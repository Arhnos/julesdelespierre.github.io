SAE 2.04 - Exploitation d'une base de données


Pour ce projet que j'ai réalisé avec Victor Dhorme. Nous avons dû exploiter d'une base de données en CSV, que nous avons du importer dans une base SQL afin de la ventiler et ainsi faire des requêtes sur des critères demandés.

Résultat de la requête suivante : "Nombre de participants sur chaque jeu olympique"


Description

Nous avions eu un fichier CSV de 68 000 lignes concernant la participation d'athlètes à des épreuves de tous les Jeux Olympiques sur la période de 1920 à 2016.

L'objectif de ce projet était de rendre ce fichier exploitable pour en tirer des statistiques.

Nous avons du dans un premier temps : analyser le fichier CSV afin de pouvoir l'importer dans une base de données. Puis nous avons ventiler les données pour réduire la taille de la base.

Puis ensuite nous avons faire des requête pour faire un rapport de statistiques.

Date

Projet réalisé en avril 2023 (1 ère année de B.U.T. Informatique)

Compétences

  • Travail en binôme
  • Récupérer des données à partir d'un fichier
  • Importer des données dans une base de données
  • Ventiler des données
  • Faire des requêtes sur des tables
  • Interpréter des données
